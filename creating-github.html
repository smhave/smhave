GitHub

GitHub is a website where coders can store their local repository of version control using git, on the web. Users can share their code with other users, and can even branch off and collaberate with other users.

At first I was following along to Rio on the Class Topics video, and I made the mistake that she did, which was naming the new folder the same as the parent (in the video her’s was riofccstudent/riofccstudent). But then before I finished the set-up I took a look on the hands on page and noticed she said to wait until the end of the video to set it up. In the video, to correct her mistake, she made a new directory and said she will delete the old directory. Instead of doing that, I just renamed the directory and put “.github.io (Links to an external site.)" at the end of the name it already had.

Like Rio ended up doing, I opened my editor and copy and pasted the skeleton HTML from the newly created HTML document into the GitHub page, and made an h1 heading of “Hello World.” That part was pretty easy.
