UI Design

As websites and web apps have become more complex, the single role of a web designer is no longer sufficient to create a great web app/site. There are many different skill sets involved in creating web apps/sites that it is not feasible for one web designer to encompass them all. Now there are UI designers (user interface designers) and UX designers (user experience designers), and it is important to know the different between the two roles.

User interface design (UI design) focuses on improving look, feel, presentation, and interactivity of the web app/site. Visual and front end design is the primary focus of the UI designer. This focus is applied to the pages the end user uses to interact with the web app/site. The UI designer also ensures that the visual layout of the web app/site adequately guides the end user on how to appropriately navigate the app/site.
