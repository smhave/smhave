Steve's Project 0

Site Info

I am Steve; welcome to my website. This is Project 0, my final project for the CIT 82 Web Technologies course at Fresno City College. Here on this site you will find information on various web development related topics that I learned about in this class and that I found interesting. This site was built based upon a free bootstrap template called Multi that I found on the web.

One thing I liked about the Multi bootstrap template is that it came with a drop down menu item built into the navigation bar. I ended up using that code for all of my menu items. This website features twenty different topics, each with its own page, and each featured in one of five different categories; so I figured that instead of having separate pages for each menu item with links to each sub-menu category on its relevant menu page, I would link to each page in the navigation bar through drop down menus on each and every page. It took sort of a long time to test every link in the navigation bar in all twenty pages, but I like how it looks and I think it turned out well.
